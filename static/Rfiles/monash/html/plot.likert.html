Likert-scale data plot

Description

Plots percentages or numbers of likert-scale data

Usage

## S3 method for class 'likert':
plot(x, labels = x$labels, percent = FALSE, col = "grey", xlab = ifelse(percent, "Percent", "Number"), ...)
## S3 method for class 'likert':
boxplot(x,...)

Arguments

x An object of class likert
labels Labels used in Likert-scale data
percent A logical value, if FALSE (by default), it plots frequencies of Likert-scale data, otherwise percent of data
col The colour for the bars
xlab A title for the x axis
... Additional arguments to plot

Value

None

Author(s)

Rob J Hyndman

Examples

x <- c(rep(3,9),rep(4,7),rep(5,14))
y <- likert(x)
plot(y,percent=TRUE)
boxplot(y)

# Side-by-side boxplots
y2 <- likert(sample(1:5,20,TRUE))
boxplot(list(y,y2))
