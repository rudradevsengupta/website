table.likert {monash}R Documentation

Likert-scale data table

Description

Creates tables of Likert-scale data

Usage

## S3 method for class 'likert':
table(x,...)
tablepc(x, sort = FALSE,...)

Arguments

x An object of class likert
sort A logical value, if TRUE, it sorts labels by percent
... Other arguments

Value

Table

Author(s)

Rob J Hyndman

See Also

See Also likert

Examples

## Let x be a set of responses to 5-point Likert scale questions
x <- c(rep(3,9),rep(4,7),rep(5,14))
y <- likert(x)
table(y)
tablepc(y)
